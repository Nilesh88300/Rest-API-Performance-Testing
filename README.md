# REST API & Performance Testing Automation

A comprehensive REST API testing and performance testing project demonstrating functional API validation, regression testing, and load testing using **Postman** and **Apache JMeter**.

---

##  Project Overview

This project showcases practical API testing and performance testing techniques by validating REST APIs, automating API checks, and analyzing application performance under concurrent user loads.

The project includes:

- Functional API Testing
- Regression Testing
- API Automation
- Load Testing
- Performance Testing
- HTML Dashboard Reporting

---

##  Features

- REST API Functional Testing
- HTTP Method Validation (GET, POST, PUT, DELETE)
- Request Header Validation
- Authentication Testing
- JSON Response Validation
- HTTP Status Code Verification
- Automated API Test Scripts
- Performance Testing with Apache JMeter
- Load Testing
- Concurrent User Simulation
- HTML Performance Report Generation
